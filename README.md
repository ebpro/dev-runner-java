# Dev Runner Java Docker Image

This Docker image is designed for use in GitHub Actions and provides a Debian-based environment with pre-installed Java development tools, including Java, Kotlin, Gradle, and Maven. It leverages SDKMAN for managing software development kits, ensuring that the latest versions of these tools are readily available for Java development projects.

## Features

- **Pre-installed Java Development Tools**: Comes with Java, Kotlin, Gradle, and Maven installed via SDKMAN, making it ready to use for Java development out of the box.
- **GitHub Actions Runner**: Compatible with GitHub Actions, allowing for seamless CI/CD integration.
